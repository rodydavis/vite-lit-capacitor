## Getting Started

To build the web folder and sync the assets:

```bash
npm run build
```

To run ios or android:

```bash
npm run start:ios
npm run start:android
```

To run desktop:
```bash
npm run open:desktop
```

To run web:

```bash
npm run dev
```
